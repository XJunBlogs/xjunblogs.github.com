发表于 2017-10-18 | 分类于 前段

jquery 概念:

JQuery是一个轻量级的JS的框架（JS的类库）.对传统的JS进行了封装.
现在企业的开发中往往不会使用传统的JS进行开发，通常都会使用JS框架进行开发.
JS的常用的框架:JQuery,ExtJS,DWR,Prototype...
jQuery的引用方式
jQuery的引入方式:<script src="../js/jquery-1.11.3.min.js" type="text/javascript" charset="utf-8"></script>
jQuery和js对象的相互转化
jQuery转js
1  var $id=$("#id")//声明一个jQuery对象
   $id[0].innerHTML=""// 将jQuery转化为js对象,并调用innerHTML
2  $id.get(0).innerHTML="" //方式二
js转jQuery
1  var id=document.getElementById("id");
   $(id).html("")//将jquery转化js对象
jquery的事件
概念:
因为jquery的事件和js的事件声明不同,不像js的事件大多都是写在便签属性中.
所有jquery要想触发事件,要先写$(function(){要执>>行的jquery代码,通常用来绑定事件});
作用和onload=function(){};相同

$(function(){})和onload=function(){}的比较:
1  $(function()等页面的DOM树绘制完成后就能进行执行
2  可以多次执行,且效率高
Jquery的效果
1  show()和hide()     
  show显示:通过jquery对象调用show()可以让隐藏的元素显示同时show()中也可以传入一些参数去控制元素从无到有显示的时长
  hide隐藏:使用的方式和show()一样
  方式一:show()  hide()
  方式二:show(毫秒值)  hide(毫秒值)
  方式三:show(["slow","normal","fast"])  hide(["slow","normal","fast"])三个元素选一个

2  向下和向上效果的显示与隐藏(使用同上)
   slideDown()方法和 slideUp()方法

3  淡出淡出的显示与隐藏的效果(使用同上)
   fedeOut()方法和fedeIn()方法

jquery的选择器

基本选择器
ID选择器: $("#id")
类选择器: $(".类名")
元素选择器: $("元素名称")
通配符选择器:$("*")
并列选择器 格式:$("选择器,选择器,选择器"):
层级选择器
空格:元素的后代元素 
>:子元素选择器
+:下一个兄弟(同辈)元素
~(波浪线):后面的所有的兄弟(同辈)元素
过滤选择器
特点:以:号开头,通常和其他选择器搭配使用,用于过滤元素

基本:
:first 获得匹配元素集合中的第一个元素
如:$("div input:first")获得div下所有的input表单中的第一个元素

:last  获得匹配元素集合中的最后一个元素
如:$("div input:last")获得div下所有的input表单中的最后一个元素

:odd   获得匹配元素集合中索引为奇数的元素,从0开始

:even  获得匹配元素集合中索引为奇数的元素,从0开始

:eq(index) 获得匹配元素集合中指定索引的元素

:lt(index) 获得匹配元素集合中,索引比给定索引小的元素


内容:
:contains(text) 匹配包含给定文本的元素

:parent 匹配含有子元素或者文本的元素

:has(selector)匹配含有选择器所匹配的元素的元素 
 如: $("div:has(p)").addClass("test"):
 给所有包含p元素的div元素添加一个text类
属性选择器
$("元素[属性]")
 匹配所有含有指定属性的元素,如$("div[id]")
$("元素[属性='值']
匹配所有属性='值'的元素
表单选择器
:input 匹配所有input textarea,select和button元素
:text  匹配所有的文本框
:password 匹配所有的密码框
:radio  匹配所有的单选按钮
:checkbox 匹配所有的多选按钮
:submit 匹配所有的提交按钮
:image 匹配所有的图像域
:reset 匹配所有的重置按钮
:button 匹配所有的按钮
:file 匹配所有的文件域
:hidden 匹配所有的不可见元素,或者type为hidden的元素
表单对象属性
:enabled 匹配所有可用元素
:disabled 匹配所有不可用元素
:checked 匹配所有点击过的元素(复选框,单选框等,不包含select中的option)
:selected 匹配所有选中的option元素
JQ的属性操作
attr(name|pro|key,val|fn)
设置或返回被选元素的属性值，只能获取初始值，并且用户改变后，使用该方法无法再次修改

removeAttr(name) 
从每一个匹配的元素中删除一个属性

prop(name|pro|key,val|fn)
获取在匹配的元素中的第一个元素的属性值

removeProp(name)
用来删除由.prop()方法设置的属性集

css属性:
addClass(class|fn)为每个匹配的元素添加指定的类名
removeClass(class|fn)从所有匹配的元素中删除全部或者指定的类
文档处理
内部插入
append(content|fn) 向每个匹配的元素内部追加内容
appendTo(content)  把所有匹配的元素追加到另一个指定的元素元素集合中

外部插入
insertAfter(content)把所有匹配的元素插入到另一个、指定的元素元素集合的后面
insertBefore(content)把所有匹配的元素插入到另一个、指定的元素元素集合的前面
After()
before()

删除
remove([expr])从DOM中删除所有匹配的元素
JQuery的CSS操作：
css(name|pro|[,val|fn])
访问匹配元素的样式属性
JQuery的事件切换
toggle([speed],[easing],[fn]);--单击事件的切换(1.9以上版本不支持需要添加插件)
用于绑定两个或多个事件处理器函数，以响应被选元素的轮流的 click 事件。

hover([over,]out)
一个模仿悬停事件（鼠标移动到一个对象上面及移出这个对象）的方法
over:鼠标移到元素上要触发的函数
out:鼠标移出元素要触发的函数
HTML代码/文本/值
html([val|fn])取得第一个匹配元素的html内容
text([val|fn]) 结果是由所有匹配元素包含的文本内容组合起来的文本。
val([val|fn|arr]) 获得匹配元素的当前值。
JQuery的遍历
方式一:
JQ的对象.each(function(i,n){}); //i为索引,n为遍历例的值
例:
以每一个匹配的元素作为上下文来执行一个函数。
$("img").each(function(i){
this.src = "test" + i + ".jpg";
});

方式二:
$.each(数组,function(i,n){});
例:
通用例遍方法，可用于例遍对象和数组。
$.each( [0,1,2], function(i, n){
alert( "Item #" + i + ": " + n );
});
