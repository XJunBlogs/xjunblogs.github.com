统计网站登录的人数

分析:使用ServletContext的存取数据的功能

ServletContext的作用:
1. 用来获得全局的初始化参数
2. 用来获得文件的MIME的类型
3. 作为域对象存取数据
4. 读取web下的文件
作用范围:整个web工程
创建:服务器启动时,Tomcat服务器会为每一个web项目提供一个ServletContext
销毁:当项目从服务器移除时,或服务器关闭时ServletContext会销毁

步骤:
1. 在登录的LoginServlet中重写init方法,获得ServletContext对象,设置属性键为count,值为0
2. 在web.xml中LoginServlet的Servlet标签中设置load-on-startup标签值为2,让LoginServlet在服务器启动的时候初始化
3. 当在LoginServlet中检查到登录成功时,获得ServleContext对象拿到count的值,让它加一,然后重新设置回去
4. 然后在需要显示人数的界面,拿到count的值就行了
关键代码实现:
init方法的实现
@Override
public void init() throws ServletException {
    // TODO Auto-generated method stub
    getServletContext().setAttribute("count", 0);
}

web.xml中的设置指定Servlet的设置
<servlet>
<description></description>
<display-name>LoginServlet</display-name>
<servlet-name>LoginServlet</servlet-name>
<servlet-class>servlet.LoginServlet</servlet-class>
<load-on-startup>2</load-on-startup>
</servlet>

Servlet中代码的实现
在登录成功的情况下:
response.getWriter().print(" 登录成功");
 //拿到登录人数
 int count = (Integer) getServletContext().getAttribute("count");
 //加一
 count++;
 //在存到域对象ServletContest中
 getServletContext().setAttribute("count", count);
 response.getWriter().println("登录成功5秒后跳转界面");
 response.setHeader("refresh", "5;url=/day12/CountServlet");
